## What is noboil?

noboil is a fullstack framework that generates your entire backend from a Zod schema.
Define your tables once — get CRUD, real-time subscriptions, file uploads, typesafe forms, multi-tenancy, and more. All type-safe, all zero-config.

Currently supports:

- **Convex** — Document store with automatic caching, optimistic updates, and serverless functions
- **SpacetimeDB** — Relational database with WebSocket subscriptions and Rust-powered modules

The consumer code is nearly identical across databases — switch by changing one import.

## Quick Start

```bash
bun noboil@latest init
```

Choose your database, pick your demo apps, and you're shipping in minutes.

## Features

- **Schema-first** — Define tables with Zod, get everything else generated
- **Zero-boilerplate** — No hand-writing CRUD, pagination, or auth boilerplate
- **Type-safe** — End-to-end types from database to UI, including Swift codegen
- **Real-time** — Live subscriptions out of the box
- **Multi-tenancy** — Organizations, roles, invites, ACL in one function call
- **Forms** — Auto-generated form components with validation
- **File uploads** — Drag-and-drop with compression and progress
- **Soft delete** — Built-in with restore and permanent delete
- **DevTools** — Schema playground, query inspector, subscription monitor
